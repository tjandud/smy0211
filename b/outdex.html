가상의 인터넷 안전 결제(ISP)의 비밀번호만으로도 안전한
결제 서비스를 이용하실 수 있습니다.		 소유하신 카드에 미리 설정해 둔 전자상거래용의
별도 비밀번호를 입력하여 카드 사용자 본인을 확인 한 후
결제 서비스를 이용하실 수 있습니다.

신청 및 이용안내
  1. 배송/결제 페이지 결제 수단 선택 영역 에서
    "신용카드", "할부기간" 선택 후 [결제하기] 클릭
  2. 안전결제(ISP) EXE 프로그램 설치 (최초 1회)
  3. 안전결제(ISP) 서비스 결제창 카드 선택 영역의
    [카드 등록]버튼 클릭
  4. 신용카드 번호와 E-mail 주소를 입력 후 [다음] 클릭
  5. 신용카드 비밀번호, CVC코드, 유효기간 입력 후 [다음]
  6. 안전결제(ISP) 비밀번호를 입력 후 [확인] 클릭
  1. 배송/결제 페이지 결제수단 선택 영역에서 "신용카드",
    "할부기간" 선택 후 [결제하기] 클릭
  2. 안심클릭 카드사별 EXE프로그램 설치(카드사별 최초 1회)
  3. 안심클릭 등록 (최초 안심클릭 서비스 이용 시)
    • ⋅카드번호 입력 후 [확인] 클릭
    • ⋅주민등록번호/ CVC/ 비밀번호 입력 후 [확인] 클릭
    • ⋅휴대폰 인증 혹은 공인인증 이후 [확인] 클릭
    • ⋅이용약관에 동의 후 개인확인 메시지와
        안심클릭 비밀번호 등록
  4. 안심클릭 결제
    • ⋅카드번호, 안심클릭 비밀번호, CVC코드 입력 후 결제.

* 안심클릭 서비스 결제창 내의 "간편결제 서비스" 및
  "앱카드결제 서비스"를 선택하시면 정보 등록 후 간편하게
  이용하실 수 있습니다.
